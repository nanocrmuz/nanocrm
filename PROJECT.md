# Nano CRM Landing — Описание проекта

Полное руководство для разработчиков, работающих над лендингом Nano CRM.

---

## Обзор проекта

**Nano CRM** — SaaS-продукт для частных мастеров (маникюр, педикюр и т.д.) в Узбекистане. Система автоматизирует запись клиентов через Telegram-бота, работает 24/7 и помогает мастерам выделяться среди конкурентов.

**Лендинг** — одностраничный сайт (single-page landing) с презентацией продукта, ценами и призывами к действию.

## Архитектура

### Технический стек
- **HTML5** — семантическая разметка
- **CSS3** — все стили inline в `<style>` теге
- **Vanilla JavaScript** — минимальный JS для анимаций и слайдера
- **Google Fonts CDN** — Playfair Display, Inter, JetBrains Mono
- **Никаких зависимостей** — нет npm, webpack, React, Vue и т.д.


**Темная тема (для карточек):**

- `#1F1218` — вторичный темный (dark-2)

### Типографика

**Шрифты:**
1. **Playfair Display** (serif) — заголовки h1, h2, h3, логотип, цены
2. **Inter** (sans-serif) — body текст, UI элементы, кнопки

**Размеры заголовков (responsive):**

**Веса шрифтов:**
- h1, h2: 600 (semibold)
- h3: 500 (medium)
- body: 400 (regular)
- кнопки, UI: 500-600

**Line-heights:**
- h1: 0.98 (очень плотно)
- h2, h3: 1.05
- body: 1.55
- feature-pill strong: 1.2
- feature-pill span: 1.2

### Система отступов

**Все отступы кратны 4px:**
```
4, 8, 12, 16, 20, 24, 28, 32, 36, 40, 44, 48, 60, 64, 80, 100, 140
```

**Ключевые отступы:**
- Между секциями: 140px (десктоп), 100px (мобилка)
- От section-head до контента: 48px
- Боковые отступы wrap: 24px (десктоп), 16px (мобилка)
- Gap в bento: 20px (десктоп), 16px (мобилка)
- Padding карточек: 32px (десктоп), 24px (мобилка)

### Радиусы скругления
- `--radius`: 24px (основной)
- `--radius-lg`: 32px (большой)
- `--radius-sm`: 16px (маленький)
- Кнопки: 999px (полностью круглые)

## Как редактировать

### Изменить текст
1. Найти нужную секцию в HTML
2. Изменить содержимое тегов `<h1>`, `<h2>`, `<h3>`, `<p>`, `<span>`, `<strong>`
3. Сохранить и обновить браузер

### Изменить цвета
1. Найти `:root` в начале `<style>`
2. Изменить CSS-переменные
3. Все цвета обновятся автоматически

### Добавить секцию
1. Скопировать существующую секцию
2. Изменить контент
3. Убедиться что используются правильные классы bento grid

### Заменить изображения
1. Положить новые изображения в `assets/`
2. Обновить `src` в тегах `<img>`
3. Проверить размеры и пропорции

### Изменить отступы
Все отступы должны быть кратны 4px. Используйте значения:
```
4, 8, 12, 16, 20, 24, 28, 32, 36, 40, 44, 48, 60, 64, 80, 100, 140
```

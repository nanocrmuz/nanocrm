## 02. Шрифты, цвета, CSS-переменные

### Typography (Google Fonts)
- `Playfair Display` — все заголовки h1–h3, акценты
- `Inter` — body, UI, кнопки
- `JetBrains Mono` — README, технические подписи

### Размеры заголовков
```css
h1: clamp(48px, 8vw, 82px)    // от 48px (мобилка) до 82px (десктоп)
h2: clamp(36px, 5vw, 54px)    // от 36px до 54px
h3: clamp(24px, 3.5vw, 32px)  // от 24px до 32px, font-weight: 500
```

### Отступы между секциями
- **Десктоп**: 140px между большими блоками
- **Мобилка (≤560px)**: 100px между блоками
- **Боковые отступы**: 24px (десктоп), 16px (мобилка)
- **Отступ от section-head до контента**: 48px

### Система отступов
Все отступы кратны 4: 4px, 8px, 12px, 16px, 20px, 24px, 28px, 32px, 36px, 40px, 44px, 48px, 60px, 64px, 80px, 100px, 140px

```css
--ink-soft:     #4A3640;
--muted:        #8C7884;

--accent:       #C0395A;   // розово-бордовый акцент
--accent-deep:  #9E2A47;

--radius:       24px;
```

## 03. Описание секций и содержимого

- **Hero** — крупный заголовок-антиква с курсивным акцентом, badge «Ранний доступ · Скидка 36%», CTA + side-карточка со скриншотом интерфейса + 3 feature-pill карточки.
- **Проблемы** — 4 одинаковых блока 2×2 с римскими номерами в иконке.
- **Решение** — 6 фич: одна большая accent-карточка, 4 равных, плюс широкая тёмная плитка с CTA «Связаться с менеджером».
- **Скрины** — два ряда реальных скриншотов. На десктопе: сетка 2 колонки, на мобилке: горизонтальный слайдер с навигацией.
- **Demo CTA** — pink-карточка с кнопкой «Попробовать демо».
- **Цена** — слева 6 чек-листов в bento-сетке, справа тёмный price card: 800 000 → 500 000 сум / 365 дней.
- **Final CTA** — тёмный блок на всю ширину.
- **README** — этот блок: тёмный фон, моноширинный, аккордеон.

## 04. Воссоздание с нуля

### Зависимости
- Никаких npm-пакетов, бандлеров, UI-фреймворков.
- Один CDN-вызов — Google Fonts (Playfair Display, Inter, JetBrains Mono).

### CDN
```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:ital,wght@0,400;0,500;0,600;0,700;0,800;1,400&family=Inter:wght@300;400;500;600;700&family=JetBrains+Mono:wght@400;500;600&display=swap" rel="stylesheet">
```

### Запуск локально
```bash
# просто открыть в браузере
open landing.html

# или поднять статикой
python3 -m http.server 8080
```

### Редактирование контента
- Весь текст — внутри тегов `<h1>`, `<p>`, `.card`. Меняется напрямую в HTML.
- Цены — секция `#pricing`, блоки `.price-old` / `.price-now`.
- Ссылки на менеджера: `https://t.me/crmnano2`
- Ссылки на демо-бота: `https://t.me/dinara_master_demo_bot`
- Замена скринов: файлы в папке `assets/`

## 05. Применённые техники

- **Bento Grid** — 12-колоночная сетка через `display:grid` + `grid-template-columns:repeat(12,1fr)`. Карточки занимают разное число колонок (`.c-3/.c-4/.c-5/.c-6/.c-7/.c-8/.c-12`) и при необходимости 2 ряда (`.r-2`).
- **CSS Grid + Flexbox** — Grid для bento, Flexbox внутри карточек для выравнивания.
- **Responsive breakpoints**:
  - `≤ 980px` — карточки 3/4 → 6, остальные → 12
  - `≤ 560px` — все карточки 12, уменьшен padding и gap
- **CSS Variables** — единый источник правды для цветов и радиусов.
- **Fluid typography** — `clamp()` для h1/h2/h3.
- **Микро-анимации** — pulse у badge, translateY у карточек на hover, IntersectionObserver для entrance-фейда.
- **Native accordion** — README на чистых `<details>` / `<summary>`, без JS.
- **Мобильный слайдер** — горизонтальный scroll с snap для скриншотов, навигация стрелками.
- **Контраст без серых заливок** — фон страницы строго белый, глубина создаётся accent / dark / pink-tint карточками и тонкими границами.
- **Доступность** — семантические теги, контраст текста ≥ 4.5:1, фокус-стили браузера сохранены.
